Le CSS dans la balise

La première façon de faire du style dans un éléments est de déclarer l'attribut style sur celui-là. Ce n'est pas la meilleure façon manière de le faire

Je suis un paragraphe styléééééééé !

Le CSS dans l'en tête

Une seconde façon de faire du style sur notre page et de mettre la balise style dans la partie head. Cette méthode n'est toujours pas la plus conseillée mais elle permet plus de choses que la méthode balise en ligne, on peut désormais grâce a cette méthode viser plusieurs éléments d'un coup. Exemple, on mets du gris en fond sur le corps de la page, les h2 en rouge et les h3 avec un fond rose

Le CSS dans une feuilles de style externe

La dernière et meilleure méthode pour faire du style est la méthode suivante: on créer un dossier CSS avec un fichier style.css, à l'intérieur il faudra alors lier nos deux fichier grâce à la balise link. On pourra alors faire du style sur les éléments qui se trouvent dans notre page. Exemple: on déclare que tous les paragraphe doivent être en bleu.
