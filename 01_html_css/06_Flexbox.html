Direction

Flexbox nous permet d'agencer ces éléments dans le sens qu'on veux. Avec felx-direction, On peut les positionner verticalement ou encore les inverser. Il peut prendre les valeurs suivantes:
-> row: organisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse: organisés sur une ligne mais en ordre inversé
-> column-reverse: organisé sur une colonne mais en ordre inversé
