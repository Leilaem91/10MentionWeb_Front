C - Direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction, on peut les positionner verticalement ou encore les inverser. il peut prendre les valeurs suivantes:
-> row : organisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse : organsés sur une ligne mais en ordre inversé
-> column-reverse : organisé sur une colonne, mais en ordre inversé

D- Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1-Alignement sur l'axe principal

Afin d'aligner nos éléments sur l'axe principal on utilise la propriétée du flex dans un Justify-content

Cette propriétée gère la distribution des emplacemnts des flex-items suivant la place disponible dans le flex-container. Elle le fait selon le positionnement de l'axe principal

  • flex-start : aligner au début (par défaut)
  • flex-end : aligner à la fin
  • center : aligner au centre
  • space-between : les éléments sont étirés sur tout l'axe. Il y a de l'espace entre eux
  • space-around : Les éléments sont étirés sur tout l'axe, mais il laisse aussi de l'espace sur les extrémités
  • space-evenly : les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal

  • Flex-end : aligner a la fin
  • Flex-center : aligner au centre

E- Retour à la ligne

Par défaut les blocs essaient de rester sur la même ligne, même si ils n'ont pas la place (ce qui provoque parfois des problèmes de conception). Si vous le souhaitez, vous pouvez demandez aux blocs de passer à la ligne lorsqu'ils n'ont plus de place en utilisant la propriété flex-wrap

Elle peut prendre les propriétées suivantes:

  • nowrap: Pas de retour à la ligne ( la valeur par défaut)
  • wrap: Les éléments vont à la ligne lorsqu'il n'ont plus de place
  • wrap-reverse : Les éléments vont à la ligne lorsqu'il n'ont plus de place en sens inverse

F - Répartir plusieurs lignes

Si vous avez plusieurs ligne dans votre flexbox,vous pouvez choisir comment celles-ci seront réparties avec align-content. Cette propriété permet de gérer l'alignement des différents lignes (ou colonnes) les uns par rapports au autres. Cette propriété n'aura d'éffet que si le conteneur flexible posséde plusieurs lignes(si l'axe principale est horizontal) ou plusieurs colonnes ( si l'axe principle est vertical) et que si le flex-wrap est présent.

Les valeurs de cette propriété :

  • flex-start : les éléments sont placés au début
  • flex-end : les élélements sont placés à la fin
  • center : les éléments sont placés au centre
  • space-between : les éléments sont séparés avec de l'éspace entre eux
  • space-around : les éléments sont séparés avec de l'éspace entre eux, mais il y'a aussi de l'éspace au début et à la fin
  • space-evenly : Les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal.
  • stretch (par défaut) : les éléments s'étirent pour occuper tout l'éspace
